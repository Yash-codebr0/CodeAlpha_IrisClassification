# Iris Flower Classification

## Objective
Build a machine learning model that classifies iris flowers into species using their measurements.

## Dataset
Built-in Iris dataset from Scikit-learn.

Features:
- Sepal length
- Sepal width
- Petal length
- Petal width

Target:
- Setosa
- Versicolor
- Virginica

## Model Used
Random Forest Classifier

## Steps Performed
1. Loaded dataset
2. Split into training/testing sets
3. Trained model
4. Predicted results
5. Evaluated accuracy
